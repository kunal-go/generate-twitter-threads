# Installation

1. Clone the repo
    ```sh
    git clone https://github.com/kunal-go/generate-twitter-threads
    ```
2. Set appropriate environment variables

NODE_ENV<br />
PORT<br />

3. Install NPM packages
    ```sh
    npm install
    ```
4. Run

    ```sh
    npm run dev

    ```

5. Run Test cases
    ```sh
    npm test
    ```
